个人项目列表

以下为我托管在 GitHub 上的项目列表，主要为一些练手的小应用。使用 C/C++、Python 2.7/3.5 和 Golang 1.7.3 编写。项目列表将持续更新。如果你有任何想法或建议，欢迎来信或提交你的 PR，能获知你的意见是我的荣幸。

The English Version Is Here

  • 汇编语言基础练习题，收录学习汇编语言过程中解决的习题，你可以向此仓库分享你的代码，或者加入你自己的练习题。
  • 一个 Windows 平台的远程控制软件，提供了基本的图形交互界面，使用 C++ 编写。
  • 简陋的DLL注入工具，使用 C++ 编写。该仓库同时提供了一个 x64 的DLL模板，你可以修改模板为你要注入的 DLL。
  • 为北邮学生设计的脚本，包括：自动网关登陆、自动登陆URP并计算GPA等。这些脚本仅可在北京邮电大学校内使用。传送门。
  • 一个 txt 播放器，将一个文件夹中的 txt 文件在终端中播放。你可以使用这个工具 将一个视频文件转换为一系列用符号表示的txt文件。传送门。
  • 视频聊天工具，同时支持 IPv4 和 IPv6，目前的版本允许跨协议建立连接，即允许两个客户端选择自己要使用的 IP 协议，例如校内用户可以使用 IPv6 和校外的 IPv4 用户建立连接，但需要一个同时支持 IPv4 和 IPv6 的主机部署服务器。
  • 在线视频下载工具，给定视频播放的网页，通过模拟行为获取视频真实链接并下载视频，经测试 bilibili 可行。
  • 端口扫描工具，使用三种语言编写（C++，Python，Golang），性能有差异。
  • 图片为载体的信息读写库，提供了将信息写入图片的接口。
  • 运动检测监控记录工具，一个类似普通监视器管理的程序，三种模式显示给监控者，背景减除容易发现监控异常，检测到运动后开始记录。
  • 马尔科夫链随机语言生成库，使用 Python 2.7 编写，提供了基于给定的训练数据生成中/英文随机文本的接口。
  • 文本共现示例，从普通文本中捕获共现关系并提取为可视化网络。
  • 文件传输工具，使用 Golang 编写，局域网/固定公网IP 可使用。
  • 云存储系统，使用 Golang 1.7.3 编写，将在两个月内完成。它包含一个可供设计基本云存储系统的包，同时将提供一个 Windows 10 版本（可能会有安卓版本）的客户端。你可以在这里查看我在编写此项目过程中发布的专栏。

项目列表正在持续更新中…
